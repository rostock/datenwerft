# *Stadtbereichskatalog* → Bereich *Daten* → Löschung

Auf dieser Seite haben Sie die Möglichkeit Daten zu **löschen.**

## Ablauf

1. **Schema auswählen:**
   In welcher Domäne befinden sich die zu löschenden Inhalte (Faktendaten)?
   Entsprechend muss das passende Datenbankschema gewählt werden.
2. **Tabelle auswählen:**
   Welche konkrete Quelle umfasst die zu löschenden Inhalte (Faktendaten)?
   Entsprechend muss die passende Datenbanktabelle
   innerhalb des zuvor selektierten Datenbankschemas gewählt werden.
3. **Filter setzen** – siehe dazu nachfolgenden Abschnitt
4. auf Button ***löschen* klicken,** um entsprechende Datenlöschung durchzuführen
5. Es erscheint ein Dialog, der entweder über den Erfolg oder über
   den Misserfolg der Datenlöschung berichtet – in letzterem Fall werden dann die Fehlermeldungen
   aufgelistet.

## Filter setzen

Je nach gewähltem Schema und/oder gewählter Tabelle werden verschiedene Filter
aktiviert, von denen **mindestens ein Filter** verwendet werden **muss** –
zum Beispiel ein Filter auf das Jahr.

**Hinweis:** Es muss hier mindestens ein Filter verwendet werden, damit nicht (versehentlich)
zu viele Daten gelöscht werden.
